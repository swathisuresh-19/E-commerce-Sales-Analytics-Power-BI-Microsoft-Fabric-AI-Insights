
📌Project Overview

This project demonstrates an end-to-end Business Intelligence workflow by transforming raw e-commerce sales data into interactive dashboards and AI-powered business insights. The project was completed in three stages using different analytics platforms: Google Looker Studio, Microsoft Power BI, and Microsoft Fabric.

The objective was to analyze sales performance, profitability, customer purchasing behavior, regional trends, and product performance while leveraging AI features to support data-driven decision-making. Each platform builds upon the previous stage, showcasing progressive analytical capabilities from data visualization to AI-assisted insights.

📌Tools Used

Microsoft Power BI Desktop — Data modeling, DAX measures, KPI creation, and dashboard design
Power Query — Data transformation and preparation
Microsoft Fabric — Lakehouse, Dataflow Gen2, Semantic Model, and cloud-based analytics
Power BI AI Visuals — Smart Narrative, Key Influencers, and AI-powered business insights
Dataset — Superstore Sales Dataset 

Project Phases

📍 Phase 1 — Power BI Sales Analytics Dashboard
Imported the Superstore dataset into Power BI Desktop.
Cleaned and transformed data using Power Query.
Created DAX measures for:
Total Sales
Total Profit
Total Orders
Profit Margin (%)

Designed an interactive dashboard with:

KPI Cards
Monthly Sales Trend
Sales by Category
Profit by Region
Geographic Sales Map
Customer Performance Table
Added interactive slicers for Region, Year, Category, and Sub-Category.

📍 Phase 2 — Microsoft Fabric Analytics
Created a Microsoft Fabric Workspace.
Built a Lakehouse and uploaded the Superstore dataset.
Loaded data into Delta Tables.
Performed cloud-based transformations using Dataflow Gen2.
Created a Semantic Model with business measures.
Connected Power BI reports to the Fabric Lakehouse.
Published reports for cloud-based analytics and collaboration.

📍 Phase 3 — AI Business Insights Dashboard
Created an AI-powered insights page using Power BI AI features.
Added Smart Narrative to automatically summarize dashboard performance.
Configured Q&A Visual to support natural language queries.
Used Key Influencers to identify factors affecting profit and sales.
Added Decomposition Tree for drill-down analysis across business dimensions.
Developed AI-driven summaries to support faster business decision-making.

📌Dashboard Metrics

Total Sales: $2.30M
Total Profit: $286.40K
Total Orders: 10K
Profit Margin: 12.47%

📌Key Insights

Achieved $2.30 million in total sales with over 10,000 customer orders.
Technology is the highest-performing category in terms of revenue.
The West region contributes the highest sales and profit.
Sales performance reaches its peak during November, indicating strong seasonal demand.
The business maintains a 12.47% profit margin, reflecting healthy profitability.
AI-powered analysis helps identify trends, performance drivers, and business opportunities.
Interactive dashboards enable users to analyze performance by Region, Category, Year, and Sub-Category.
Natural language Q&A provides instant answers to business questions without requiring manual filtering.
